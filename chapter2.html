Chapter 2: Number System, Conversion, and Boolean Logic

Introduction

In this chapter, we will explore the fundamental concepts of the number system, number conversions, and Boolean logic, which are the building blocks of digital systems and computers.

Number Systems

A number system is a writing system for expressing numbers; it is a mathematical notation for representing numbers in a consistent manner using digits or other symbols. Common types of number systems include:

  • Binary System: Base 2, uses digits 0 and 1.
  • Octal System: Base 8, uses digits from 0 to 7.
  • Decimal System: Base 10, uses digits from 0 to 9.
  • Hexadecimal System: Base 16, uses digits 0-9 and letters A-F.

Number System Conversion

Number system conversion refers to the process of converting a number from one base to another. Here's how to convert between different number systems:

Binary to Decimal

To convert a binary number to decimal, multiply each bit by 2 raised to the power of its position and sum the results. For example:

                1011 (binary) = 1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 1 * 2^0
                             = 8 + 0 + 2 + 1 = 11 (decimal)

Decimal to Binary

To convert a decimal number to binary, repeatedly divide the number by 2, recording the remainders, and read the remainders from bottom to top. For example:

                11 (decimal) รท 2 = 5 remainder 1
                5 รท 2 = 2 remainder 1
                2 รท 2 = 1 remainder 0
                1 รท 2 = 0 remainder 1
                11 (decimal) = 1011 (binary)

Binary to Hexadecimal

To convert a binary number to hexadecimal, group the binary digits into sets of four (starting from the right) and convert each group into its hexadecimal equivalent. For example:

                101101 (binary) = 0010 1101
                                = 2 D (hexadecimal)

Boolean Logic

Boolean logic is the basis of digital circuits and is used for decision-making in computers. It uses variables that can only have two values: true (1) and false (0). The basic operations of Boolean logic are:

  • AND: The result is true if both operands are true. (A AND B)
  • OR: The result is true if at least one operand is true. (A OR B)
  • NOT: Inverts the value of the operand. (NOT A)
  • XOR: The result is true if one operand is true and the other is false. (A XOR B)

Boolean expressions can be simplified using laws such as:

  • Identity Law: A AND 1 = A, A OR 0 = A
  • Null Law: A AND 0 = 0, A OR 1 = 1
  • Complement Law: A AND NOT A = 0, A OR NOT A = 1
