Chapter 6: Web Technology - I

Introduction to Web Technology

Web technology refers to the tools, languages, and techniques used to create websites and web applications. In today's digital age, web technologies are an integral part of almost every business and daily task, from e-commerce to social media, entertainment, and much more.

In this chapter, we will explore the basic building blocks of web technology, including HTML, CSS, and JavaScript.

1. HTML - The Structure of Web Pages

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It provides the basic framework for web pages, allowing text, images, videos, and other elements to be displayed on a webpage.

Basic HTML Structure

A typical HTML document is structured with the following basic elements:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

Here, the <html> tag represents the start of the document, the <head> contains metadata and information like the title, and the <body> holds the content that is displayed to the user.

Common HTML Tags

  • <h1> to <h6>: Header tags used for headings (h1 is the largest, and h6 is the smallest).
  • <p>: Paragraph tag used for text.
  • <img>: Image tag used to insert images.
  • <a>: Anchor tag used to create hyperlinks.
  • <ul>, <ol>, <li>: Tags used to create unordered and ordered lists.
  • <div>: A container element used to group other elements together for styling and layout purposes.

2. CSS - Styling Web Pages

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows web designers to apply styles such as colors, fonts, spacing, and positioning to HTML elements. CSS makes websites more visually appealing and improves the user experience.

Basic CSS Syntax

The CSS syntax is quite simple and consists of selectors and declarations:

selector {
    property: value;
}

For example:

h1 {
    color: blue;
    font-size: 24px;
}

This code will make all <h1> elements appear in blue color and have a font size of 24 pixels.

Common CSS Properties

  • color: Sets the color of text.
  • font-family: Specifies the font to be used for text.
  • background-color: Sets the background color of an element.
  • margin: Adds space outside an element.
  • padding: Adds space inside an element, between its content and its border.
  • border: Defines the border style for an element.
  • display: Specifies how an element is displayed (e.g., block, inline, flex, etc.).

3. JavaScript - Adding Interactivity

JavaScript is a programming language that allows you to add interactivity and dynamic behavior to your web pages. It enables you to create features like form validation, animations, dynamic content loading, and more.

Basic JavaScript Syntax

JavaScript is typically embedded within an HTML page or linked externally. Here's an example of basic JavaScript syntax:

<script>
  alert("Hello, World!");
</script>

The above code will display a pop-up alert with the message "Hello, World!" when the page is loaded.

Common JavaScript Functions

  • alert(): Displays an alert dialog with a message.
  • document.getElementById(): Retrieves an element from the DOM (Document Object Model) by its ID.
  • console.log(): Outputs information to the browser's console (useful for debugging).
  • addEventListener(): Adds an event listener to an element (e.g., for click events, key presses, etc.).

Conclusion

In this chapter, we covered the three fundamental web technologies: HTML, CSS, and JavaScript. HTML provides the structure of a web page, CSS is used for styling and layout, and JavaScript adds interactivity and dynamic features. Understanding these technologies is the foundation of web development and will allow you to create simple static websites and dynamic web applications.
